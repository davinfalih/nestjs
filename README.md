 # Deskripsi Singkat API
API Todo List ini dibuat menggunakan NestJS tanpa database, dengan data todo disimpan dalam array. API menyediakan endpoint CRUD untuk mengelola daftar tugas.

# Daftar Endpoint
- GET /todos → Menampilkan semua todo
- GET /todos/:id → Menampilkan todo berdasarkan ID
- POST /todos → Menambah todo baru (body: { title: string, status: string })
- PUT /todos/:id → Mengupdate todo berdasarkan ID (body: { title?, status? })
- DELETE /todos/:id → Menghapus todo berdasarkan ID
